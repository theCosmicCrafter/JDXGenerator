# 🚀 JDX Generator

> **Advanced  Prompt Generation Engine for Forge, FLUX & Anima Models**

JDX Generator is a powerful  prompt generation extension for Stable Diffusion Forge. It creates coherent, highly customizable prompts by intelligently combining thousands of carefully curated tags across dozens of categories.

Instead of simply randomizing tags, JDX Generator follows a structured prompt flow to generate complete characters with logical composition and model-specific optimization.

# 📦 Installation

## Stable Diffusion Forge

1. Open Forge
2. Go to **Extensions**
3. Click **Install from URL**
4. Paste the GitHub repository URL
5. Click **Install**
6. Restart Forge
